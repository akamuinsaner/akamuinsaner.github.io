react&redux初体验

发表于 2017-08-08

前一段时间学习了react和redux，对于用惯了原生JS的我简直是无从下手，react技术栈涉及的大部分知识我都没有接触过，于是乎又学习了一下es6语法，jsx，这才开始我的react&redux之旅，学习的同时又接触到了css Module和webpack。深深地体会到了react的复杂之处。于是乎做了一个简单的待办事项单页应用以检验成果。
源码地址：https://github.com/akamuinsaner/todoList
查看演示：运行下列命令

git clone https://github.com/akamuinsaner/todoList
cd todoList
npm install
npm start

onActive(event){
   let value = event.target.checked;
   this.setState({
      active:value
   });
   this.context.store.dispatch(activeTodo(this.props.item.index));
}
复选框通过onChange监听onActive事件，只要复选框状态发生改变即向reducer发送一个含有该事项index的action，将事项的active字段改为true或false 
export const activeTodo = (index) =>{

2.todoInput
todoInput即待办事项输入组件，下面列了五个按钮（按钮的样式借鉴了bootstrap的样式），其中除了“查看”按钮之外其余都会发出一个action改变state 

 onAddButton(){
        if(this.state.whatInput.trim()){
            this.context.store.dispatch(addTodo(this.state.whatInput, this.state.whereInput, this.state.whenInput));
            this.setState({
                whatInput:'',
                whereInput:'',
                whenInput:''
            })
        }
 };
 onDelButton(){
        this.context.store.dispatch(deleteTodo());
 };

 onModButton(){
        this.context.store.dispatch(modTodo(this.state.whatInput, this.state.whereInput, this.state.whenInput));
        this.setState({
            whatInput:'',
            whereInput:'',
            whenInput:''
        })
};

onDoneButton(){
        this.context.store.dispatch(completeTodo());
};

onViewTodo(){
        let state = this.context.store.getState();
        state.map((item) =>{
            if(item.active === true){
                this.setState({
                    whatInput:item.text,
                    whereInput:item.place,
                    whenInput:item.time
                })
            }
        })
}
五个按钮分别监听通过onClick监听上述五个事件来执行相应动作 
export const addTodo = (text, place, time) =>{
    return {
        type: ActionTypes.ADD_TODO,
        text: text,
        place: place,
        time: time
    }
};

export const deleteTodo = () =>{
    return {
        type: ActionTypes.DELETE_TODO,
    }
};

export const modTodo = (text , place , time) =>{
    return {

        text: text,
        place:place,
        time:time
    }
};
